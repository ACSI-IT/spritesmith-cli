# spritesmith-cli

> Adds CLI to the [spritesmith][1] module

Spritesmith is awesome but sometimes we don't want to [run it through Grunt][2].

# Install

```shell
npm install -D spritesmith-cli
```

# Usage

```shell
spritesmith [options]
```
